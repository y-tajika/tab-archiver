# Tab Logger

Chrome/Edge拡張機能：開いているタブを一括ブックマーク

## バージョン

**v1.6.0** - 手動フォルダ名入力機能追加版

## 機能

- 🗂️ 現在のウィンドウの全タブを一括ブックマーク
- 📅 自動フォルダ名生成：`月-日_要約キーワード`
- ✏️ **NEW** フォルダ名の手動入力・編集が可能
- 🎯 保存先：「その他のお気に入り > Tab_Logger」
- ⌨️ ショートカット：**Alt+Shift+S**（自動命名で即座に保存）
- 🧠 改善された要約アルゴリズム（より意味のある名前を自動生成）

## インストール

1. このフォルダを任意の場所に保存
2. Chrome/Edgeで `chrome://extensions/` を開く
3. 「開発者モード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」
5. このフォルダを選択

## 使い方

### 2つの保存方法

#### 方法1: 手動でフォルダ名を編集（推奨）
1. 保存したいタブをブラウザで開く
2. **拡張機能アイコンをクリック**
3. ポップアップが開き、自動生成されたフォルダ名が表示される
4. フォルダ名を編集（または Enter キーでそのまま保存）
5. 「保存」ボタンをクリック

#### 方法2: 自動命名で即座に保存
1. 保存したいタブをブラウザで開く
2. **Alt+Shift+S** を押す
3. 自動生成されたフォルダ名で即座に保存される

### フォルダ名の例
- `01-15_開発環境構築_プロジェクト管理`
- `01-15_GitHubリポジトリ_ドキュメント`
- `01-15_会議資料_準備進行`

### ショートカットキーのカスタマイズ
`chrome://extensions/shortcuts` で変更可能

### 設定のカスタマイズ
1. 拡張機能を右クリック → オプション
2. 以下を設定可能：
   - 日付フォーマット（デフォルト: MM-DD）
   - 時刻の表示/非表示
   - 要約の最大文字数（デフォルト: 30文字）
   - タイトルの最大文字数（デフォルト: 100文字）
   - 保存後にタブを閉じる

## 技術仕様

- Manifest V3
- Service Worker
- ポップアップUI（350px幅、モダンデザイン）
- シンプル設計（パーミッション最小限）
- 必要なパーミッション: bookmarks, tabs, storage

## 制限事項

- `chrome://` や `edge://` などの特殊ページは保存できません
- ページコンテンツは取得しません（タイトルとURLのみ）
- Service Worker環境のため通知機能は利用できません
